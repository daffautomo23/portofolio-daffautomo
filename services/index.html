Pertanyaan yang Sering Ditanyakan

Berapa lama proses pembuatan website?
Tergantung paket yang dipilih. Landing page biasanya 3-5 hari kerja, website UMKM 7-14 hari kerja. Untuk project custom, timeline akan didiskusikan terlebih dahulu.
Apakah harga sudah termasuk domain dan hosting?
Belum. Harga yang tertera adalah untuk jasa pembuatan website saja. Untuk domain dan hosting, biayanya sekitar Rp 150.000 - Rp 300.000 per tahun tergantung provider. Saya bisa bantu setup juga.
Bagaimana cara bayarnya?
Sistem pembayaran: DP 50% di awal setelah deal, pelunasan 50% setelah website selesai dan siap publish. Pembayaran bisa via transfer bank atau e-wallet.
Apakah bisa request desain sendiri?
Bisa! Kalau Anda punya referensi desain atau mockup, kirim saja. Saya akan sesuaikan dengan kebutuhan Anda. Kalau belum ada, saya bantu dari nol juga gak masalah.
Apakah ada garansi setelah website jadi?
Ya, ada garansi minor bug fix selama 14 hari setelah website publish. Untuk maintenance rutin atau update konten, ada paket maintenance tersendiri dengan harga terjangkau.
